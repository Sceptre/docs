class sceptre.hooks.Hook(argument=None, connection_manager=None, environment_config=None, stack_config=None)¶

Bases: object

Hook is an abstract base class that should be inherited by all hooks used in hooks. Environment and stack config and the connection manager are supplied to the class, as they may be of use to inheriting classes.

Parameters

  • argument (str) – The argument in which the task can use.

  • environment_config (sceptre.config.Config) – The environment_config from config.yaml files.

  • stack_config (sceptre.config.Config) – The stack config of the stack that this parameter is associated with.

  • connection_manager (sceptre.connection_manager.ConnectionManager) – A connection_manager.

abstract run()¶

run is an abstract method which must be overwritten by all inheriting classes. Run should execute the logic of the hook.

sceptre.hooks.execute_hooks(hooks)¶

Searches through dictionary or list for Resolver objects and replaces them with the resolved value. Supports nested dictionaries and lists. Does not detect Resolver objects used as keys in dictionaries.
